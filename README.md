# Portal de Projetos IFRS Campus Ibirubá 

## O que é o Portal de Projetos IFRS Campus Ibirubá

O Portal de Projetos IFRS Campus Ibirubá é um website informativo desenvolvido com o objetivo de divulgar os projetos de ensino, pesquisa e extensão realizados no Instituto Federal do Rio Grande do Sul – Campus Ibirubá. A plataforma reúne informações sobre iniciativas desenvolvidas por estudantes e professores, permitindo que a comunidade acadêmica e o público em geral conheçam os projetos, seus objetivos e seus impactos.

O sistema foi desenvolvido utilizando apenas tecnologias web (HTML, CSS e JavaScript), proporcionando uma interface moderna, responsiva e intuitiva. O projeto busca facilitar o acesso às informações institucionais e incentivar a participação da comunidade nas atividades promovidas pelo campus.

Durante a navegação, o Portal de Projetos oferece:

* Visualização dos principais projetos desenvolvidos no IFRS Campus Ibirubá.
* Interface moderna e responsiva para computadores, tablets e smartphones.
* Cartões interativos contendo informações resumidas de cada projeto.
* Janelas modais com descrições completas, imagens e detalhes dos projetos.
* Barra de navegação com rolagem suave entre as seções da página.
* Animações que tornam a experiência de navegação mais agradável.
* Organização dos projetos por categorias, facilitando a localização das informações.
* Área destinada à apresentação do campus e das iniciativas desenvolvidas.
* Estrutura preparada para futura inclusão de notícias, eventos, galeria de imagens e novos projetos.

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (ES6)

## Objetivo

O principal objetivo do Portal de Projetos é centralizar e divulgar as iniciativas desenvolvidas no IFRS Campus Ibirubá, aproximando estudantes, servidores e comunidade externa das ações de ensino, pesquisa e extensão realizadas pela instituição.

Além de servir como um ambiente de divulgação institucional, o projeto também demonstra a aplicação prática de tecnologias de desenvolvimento web na construção de interfaces modernas e acessíveis.

## Funcionalidades

* Página inicial com apresentação do campus.
* Navegação intuitiva entre as seções.
* Cards interativos dos projetos.
* Modal com informações detalhadas de cada projeto.
* Layout responsivo.
* Animações utilizando JavaScript.
* Interface inspirada em portais institucionais modernos.
